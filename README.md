# 🪨 STONE — A Simulation of Resilience

**STONE** é um pequeno jogo experimental feito em **HTML, CSS e JavaScript puro**, onde o jogador controla algo incomum:

> **Uma pedra.**

A pedra não anda.
A pedra não pula.
A pedra não luta.

Ela **apenas existe e resiste**.

O jogo é uma **analogia sobre resiliência**, tempo e adaptação ao ambiente.

---

# 🎮 Conceito do Jogo

No mundo real, muitas vezes não podemos controlar:

* o ambiente
* as circunstâncias
* o tempo

Assim como uma pedra na natureza, o jogador precisa **se adaptar às condições ao redor**.

Durante o jogo, a pedra enfrentará diferentes ambientes:

* ❄️ Montanhas frias
* 🌊 Rios
* 💧 Cachoeiras
* 🌊 Fundo do mar
* 🧱 Subsolo profundo

Cada ambiente exige uma **forma diferente de adaptação**.

Escolher a reação correta ajuda a pedra a **resistir ao desgaste do tempo**.

---

# ⏳ Mecânica Principal

O tempo passa constantemente.

```
1 dia = 4 segundos
```

A cada dia:

* a pedra sofre desgaste natural
* o jogador deve decidir **como se adaptar ao ambiente**

Sistema de integridade:

| Evento             | Efeito |
| ------------------ | ------ |
| Passagem de um dia | -5 HP  |
| Escolha correta    | +4 HP  |
| Escolha incorreta  | -10 HP |

O jogo termina quando:

* 💥 **HP chega a 0** (a pedra quebra)
* 🏁 **30 dias são completados**

---

# 💎 Final Secreto

Se o jogador terminar o jogo com:

```
HP > 100
```

A pedra sofre uma transformação.

Depois de suportar pressão, tempo e desgaste...

Ela se torna:

**💎 UM DIAMANTE**

---

# 🎓 Tutorial Inicial

O jogo começa com um pequeno momento humorístico.

O jogador recebe instruções clássicas:

```
Use WASD ou setas para mover
Use espaço para pular
```

Mas ao tentar...

Nada acontece.

Porque **pedras não se movem**.

Esse momento reforça o conceito central do jogo:

> Nem sempre podemos mudar o mundo ao nosso redor.
> Às vezes só podemos **resistir ao tempo**.
